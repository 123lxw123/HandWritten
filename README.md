# HandWritten  

# 项目简介  

这是一个随手输入 App，实现思路是记录用户手指滑动的路径，根据手指滑动的速度计算画笔的粗细，得到笔锋的效果。输入展示方式有三种风格，展示实现思路有两种，一种是通过 RecycleView 展示，一种是通过 EditText 展示，分别对应 Master 和 Dev 分支的代码。
